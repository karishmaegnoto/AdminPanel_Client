Estimator Admin Frontend

Quick start (create-react-app was considered but project uses Vite for fast DX):

Install dependencies:

```bash
cd frontend
npm install
```

Run dev server:

```bash
npm run dev
```

Environment:
- API base URL can be provided via `VITE_API_URL` (defaults to http://localhost:5001)

Build for production:

```bash
npm run build
npm run preview
```
